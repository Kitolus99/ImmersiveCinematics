# Changelog

## [0.3.1] - 2026-05-26

### Fixed

- 修复编辑器新增关键帧后数组顺序混乱，导致 `ScriptParser` 单调递增校验失败、脚本无法加载的问题。
- 修复新建关键帧缺少 `position`、`yaw`、`pitch`、`roll`、`fov`、`zoom`、`dof` 属性，导致预览相机跳回原点的问题。
- 为关键帧修改路径添加排序保护，降低编辑器生成非法脚本的概率。

## [0.3.0] - 2026-05-22

从 0.2.0 完全重构。

### Added

- 新增完整游戏内编辑器 UI：时间轴、左侧属性面板、预览区、菜单栏。
- 新增触发器系统、触发器条件编辑器和网络同步流程。
- 新增运行时行为控制系统 `CinematicController`。
- 新增配置界面、脚本管理、HUD 拦截等管理能力。
- 新增多轨道架构：`CAMERA`、`LETTERBOX`、`AUDIO`、`EVENT`、`MOD_EVENT`。

## [0.2.0] - 2026-04-16

旧版发布版本，基于实体相机实现，后续被 0.3.x 架构替换。

- `camera/`：使用 Minecraft Entity 实现相机。
- `director/`：负责镜头序列编排。
- `script/`：解析文本脚本并驱动相机路径。
- `trigger/`：独立条件引擎，基于位置、物品、交互等触发运镜。
- `network/`：服务端触发控制和权限同步。

## [0.0.1] - 2026-01-24

- 项目初创，实现基础相机控制。
